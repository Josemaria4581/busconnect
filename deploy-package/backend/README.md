# API gestión_autobuses\n\nServidor Express para consumir la base de datos creada con \database.sql\.\n\n## Requisitos\n- Node.js 18+\n- MySQL o MariaDB ejecutando el esquema gestion_autobuses\n\n## Puesta en marcha\n1. Instala dependencias\n   `ash\n   npm install\n   `\n2. Crea el fichero .env a partir de .env.example y ajusta credenciales\n3. Inicia el servidor\n   `ash\n   npm run dev\n   `\n\nLa API se expone en http://localhost:4000/ (configurable vía PORT).\n\n## Endpoints principales\n- GET /api/health\n- GET /api/rutas\n- GET /api/autobuses\n- GET /api/conductores\n- GET /api/asignaciones\n- GET /api/mantenimientos\n- GET /api/viajes-discrecionales\n- GET /api/viajes-discrecionales/:id/chat\n- GET /api/notificaciones\n\nCada recurso admite operaciones POST, PUT/PATCH y DELETE según corresponda. Consulta el código fuente en outes/ para más detalles.\n
